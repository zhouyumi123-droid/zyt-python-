### 这一步学到了什么

你这次第一次真正碰到了 Python 里非常重要的一条规则：

“缩进不是排版问题，而是语法本身的一部分。”

也就是说：

- `for` 后面的代码必须向里缩进
- `def` 里面的代码也必须向里缩进

如果缩进不对，Python 就会直接报错。

### 当前收获

你现在已经不只是会运行脚本了，
你已经做到了：
