# 第 2 周

## 本周重点

熟悉文件、JSON 结构、异常处理和结构化输出。开始从“写一次性脚本”转向“做能接到工作流里的工具”。

## 本周练习脚本

1. `脚本_scripts/第02周_week02/material_loader.py`
2. `脚本_scripts/第02周_week02/topic_idea_generator.py`

## 本周目标

- 用正常输入、缺字段输入、坏输入分别跑这两个脚本
- 打开生成的 Markdown 和 JSON 看懂结构
- 能说清楚这两个脚本如果挂到 Coze 工具节点后，会怎么被调用

## 本周复盘问题

- 输入缺失时，一个工具应该返回什么？
- 给用户看的输出，和给调试看的输出，有什么区别？
- 你会怎么描述这两个脚本的输入字段？

## 本周产出

写一份简短笔记，至少包含：

- 一个你已经彻底看懂的字段结构
- 一个你现在已经能提前预判的失败场景
- 一个你觉得未来要改造成生产级时必须补的地方
