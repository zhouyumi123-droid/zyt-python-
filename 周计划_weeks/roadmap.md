# 12 周路线图

## 第 1-2 周

Python 生存能力：变量、字符串、列表、字典、循环、函数、JSON、文件读写、异常处理。

## 第 3-4 周

API 与自动化：`requests`、`.env`、HTTP 基础、状态码、重试、日志。

## 第 5-6 周

Coze 工作流加 Python 外部工具：输入输出字段、工具调用、调试痕迹、失败分支。

## 第 7-8 周

知识库与 RAG：切分、召回、Prompt 版本、引用检查、幻觉处理。

## 第 9-10 周

评估与稳定性：测试集、评分表、响应时间、成本、输出一致性、失败分类。

## 第 11-12 周

作品集包装：Demo 讲法、流程图、API 说明、测试报告、模拟交付。
