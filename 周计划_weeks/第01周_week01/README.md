# 第 1 周说明

这一周你不要想着“学完 Python”。

这一周只有一个目标：

先不害怕 Python。

## 这一周你到底要学什么

你只学最基础、最有用的东西：

- 什么是脚本
- 怎么运行脚本
- 什么叫输入
- 什么叫输出
- 什么叫报错
- 怎么改一点点代码再重新运行

## 这一周你会接触到的 3 个脚本

### 1. `clean_text.py`

作用：

整理文字格式。

这是你第一个要学的脚本。

### 2. `json_to_csv.py`

作用：

把一种数据格式转成另一种。

你现在先不用急着学它。

### 3. `prompt_rewriter.py`

作用：

把一段原始需求改写得更清楚一些。

这个放在你跑通第一个脚本之后。

## 这一周最重要的学习方式

不是背概念。

而是做下面这 4 件事：

1. 跑起来
2. 看结果
3. 改一点
4. 再跑一次

## 你这一周先别纠结的词

下面这些词现在先不用管：

- 列表
- 字典
- JSON
- 参数解析
- 测试

如果练习过程中遇到，我们再一点点讲。

## 这一周的最低目标

如果这一周结束时，你能做到这 3 件事，就已经合格：

1. 能运行 `clean_text.py`
2. 知道输入文件和输出文件分别是什么
3. 遇到报错时，能把报错复制出来给我

## 这一周的理想目标

如果你还有余力，再往前走一步：

1. 打开 `clean_text.py`
2. 我带你逐行看懂
3. 你自己改一个小地方
4. 再运行一次，看看结果有没有变化

## 这一周结束时你不需要达到什么程度

你不需要：

- 独立写程序
- 理解全部语法
- 背会术语
- 看懂所有文件

你只需要先迈进去。
