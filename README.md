## 这个项目里最重要的 3 个概念

### 1. Python 是什么

Python 是一种编程语言。

你可以先把它理解成：

“一种写给电脑看的做事说明书”

比如你告诉电脑：

- 打开一个文件
- 读取里面的文字
- 去掉多余空格
- 重新保存

这就是 Python 脚本在做的事。

### 2. 脚本是什么

脚本就是一个 `.py` 文件。

你可以先把它理解成：

“一个小工具”

比如：

- `clean_text.py`：清理文字格式
- `json_to_csv.py`：把一种数据格式变成另一种

### 3. Coze 和 Python 的关系

你现在可以把它理解成：

- Coze 负责“搭流程”
- Python 负责“做小工具”

以后你的很多工作，都是让 Coze 去调用 Python 工具。

## 你现在真正要学的顺序

### 第一步

学会运行 1 个最简单的 Python 脚本。

### 第二步

看懂这个脚本的输入、输出、每一行大概在干什么。

### 第三步

自己改一个很小的地方，再重新运行。

### 第四步

慢慢理解：

- 什么是变量
- 什么是字符串
- 什么是列表
- 什么是字典
- 什么是函数
