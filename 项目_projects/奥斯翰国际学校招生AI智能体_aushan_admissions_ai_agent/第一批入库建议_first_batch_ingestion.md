# 第一批入库建议

## 当前目标

先服务一个最小版本：

**奥斯翰招生短视频内容智能体 v1**

所以第一批入库资料，不追求全，
只追求：

- 可信
- 新
- 能直接做内容

## 第一批资料怎么用

### 用法 1：做内容知识底稿

把每份资料先整理成：

- 课程定位
- 适合人群
- 核心卖点
- 升学路径
- 差异化优势
- 不能乱说的点

### 用法 2：做视频选题素材池

把资料拆成可做视频的主题，比如：

- 学校办学背景
- 四大优势
- 小班精细化管理
- 多路径升学
- 港澳台联考优势
- DSE 课程适合谁

### 用法 3：做问答底稿

把家长会问的问题和资料里的标准答案对应起来。

## 第一版最推荐先做的 4 个招生内容方向

1. 学校介绍
2. 课程优势
3. 升学路径
4. 家长高频问题

## 暂时先不做的方向

- 所有素材自动剪片
- 所有课程全量数字人口播
- 所有历史资料一次性入库

这些都可以以后做，但不是现在最重要的。
