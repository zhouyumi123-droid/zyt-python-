# 内容运营自动化项目

这是你 12 周计划里的主作品集项目。

## 目标工作流

输入：

- 内容主题
- 目标受众
- 发布平台
- 语气风格
- 支撑资料

输出：

- 选题建议
- 资料摘要
- 草稿结构
- 风格检查结果
- 发布检查清单
- 复盘备注

## 分阶段建设

1. 第 1-2 周：本地 Python 小工具
2. 第 3-4 周：基于 API 的资料收集器
3. 第 5-6 周：Coze 工作流加外部 Python 工具
4. 第 7-8 周：知识库和检索层
5. 第 9-10 周：评估集和稳定性报告
6. 第 11-12 周：Demo 打包和模拟交付

- 工作流截图

- 输入输出样例
